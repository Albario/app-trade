# 📈 App Trade – Real‑Time Forex Signal Dashboard

Una dashboard moderna, veloce e mobile‑friendly per analizzare coppie Forex in tempo reale.  
L’app utilizza dati live da TwelveData e genera segnali BUY / SELL / STRONG / WEAK con una formula proprietaria ottimizzata.

---

## 🚀 Funzionalità principali

- 🔥 Segnali BUY / SELL / STRONG / WEAK in tempo reale  
- 🎨 Interfaccia dark professionale  
- 🔊 Suoni dedicati per ogni tipo di segnale  
- 🎛️ Filtri per visualizzare solo i segnali desiderati  
- ⏱️ Countdown automatico per ogni segnale  
- 🧹 Auto‑pulizia dei segnali scaduti  
- 📱 UI ottimizzata per smartphone  
- ⚡ Formula aggressiva con trend% e score dinamico  
- 🌐 Pubblicata come PWA (installabile su Android/iPhone)

---

## 📦 Tecnologie utilizzate

- HTML5  
- CSS3 (tema dark + responsive)  
- JavaScript (vanilla)  
- TwelveData API  
- PWA Manifest + Service Worker  
- GitHub Pages / Netlify

---

## 📲 Installazione (PWA)

1. Apri l’app dal browser mobile  
2. Tocca **Aggiungi alla schermata Home**  
3. L’app diventa installata come applicazione nativa
